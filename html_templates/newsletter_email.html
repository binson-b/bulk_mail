FOSSEE (Free and Open Source Software for Education) project at IIT Bombay, funded by the NMEICT, MHRD, Government of India, promotes the use of Open Source Software in Education. We have started a newsletter to keep FOSS enthusiasts abreast with the latest project activities and research trends undertaken by our teams.

The second issue of this newsletter largely focuses on Python, a general-purpose, high-level, remarkably powerful dynamic programming language, used in a wide variety of application domains. This issue includes articles on Remote assisted Python workshops, Python through Spoken Tutorials, Python Textbook companion, TBC Cloud interface, FOSSEE Forums, Yaksh - Online Evaluation Interface and SciPy India Conference.
